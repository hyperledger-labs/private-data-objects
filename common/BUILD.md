# Building the common libraries

Make sure you have environment variables `SGX_SDK`, `SGX_SSL` and `TINY_SCHEME_SRC` defined (see [BUILD.md](../BUILD.md)) and then run
```
mkdir build
cd build
cmake .. -G "Unix Makefiles" && make
```
